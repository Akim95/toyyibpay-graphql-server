# ToyyibPay GraphQL API

## 1. Clone the repository
```
git clone [repo uri]
```

## 2. Install Dependecies
```
npm install
```

## 3. Set API Key to .env file
```
API_URI=https://dev.toyyibpay.com/index.php/api/
API_KEY=your api key key
```

## 4. Run compile command or ```webpack``` command
```
npm run compile
```

## 5. Start the server
```
npm start
```
